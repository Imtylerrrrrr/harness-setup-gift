# 실습 01 — 🔭 글로벌 깔기 (5분)

> 목표: 내 컴퓨터의 **모든 프로젝트**에 적용되는 첫 세팅을 깐다.

## 왜

지금까지 AI에게 "간단하게 해줘", "쓸데없는 거 하지 마"를 매번 말로 했다면,
그 말을 **파일로 한 번만** 적어두는 것이 글로벌 세팅입니다.
이 실습이 끝나면 어느 프로젝트를 열어도 그 규칙이 자동 적용됩니다.

## 준비물

- Claude Code (또는 비슷한 코딩 에이전트) 설치되어 있을 것
- 터미널 실력: 필요 없음. 명령은 전부 Claude Code 대화창 안에서 칩니다.

## 따라 하기 — 방법 A: 플러그인 설치 (추천, 2줄)

Claude Code를 열고 대화창에 그대로 입력합니다.

```
/plugin marketplace add forrestchang/andrej-karpathy-skills
```

```
/plugin install andrej-karpathy-skills@karpathy-skills
```

끝입니다. 카파시 4원칙(코딩 전에 생각 · 단순하게 · 수술하듯 · 목표 중심)이
모든 프로젝트에 적용됩니다.

## 따라 하기 — 방법 B: 직접 파일 두기

플러그인 대신 내 손으로 관리하고 싶다면, Claude Code에게 이렇게 시킵니다.

```
이 저장소의 examples/1-global/CLAUDE.md 내용을
내 홈의 ~/.claude/CLAUDE.md 로 복사해줘. 이미 있으면 내용을 보여주고 물어봐줘.
```

> 💡 요령 — 세팅 작업 자체를 AI에게 시키면 됩니다. 숨김 폴더를 직접 찾아다닐 필요가 없습니다.

복사한 다음, 파일을 열어 **내 말로 고치는 것**이 중요합니다. 규칙이 마음에 안 들면 지우고, 내가 매번 하던 잔소리를 추가합니다.

## 확인하기

1. `/plugin` 을 입력해 Installed 탭에 플러그인이 보이는지 확인 (방법 A)
2. 아무 프로젝트에서나 새 대화를 열고, 일부러 애매한 요청을 해봅니다.
   예: "로그인 기능 추가해줘" (어떤 방식인지 말하지 않고)
3. AI가 **멋대로 정하지 않고 가정을 말하거나 물어보면** 성공입니다.

## 잘 안 되면

- 방법 A와 B를 동시에 써도 됩니다. 겹치면 둘 다 읽힙니다.
- 규칙이 적용 안 되는 것 같으면 새 대화로 다시 시작해보세요. (규칙은 대화 시작 시점에 읽힙니다)
