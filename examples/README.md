# examples — 3층 세팅, 실제 파일 모양 그대로

줌 렌즈 3층이 실제 파일로는 어떻게 생겼는지 보여주는 폴더입니다.

```
1-global/            🔭 광각 — 내 컴퓨터의 ~/.claude/ 에 넣을 것
├── CLAUDE.md            어느 프로젝트든 적용되는 범용 규칙
└── skills/make-doc/     어느 프로젝트든 쓰는 범용 스킬

2-project/           🎯 표준 — 각 프로젝트 폴더에 두는 것
├── .claude/             이 프로젝트 전용 규칙 + 맞춤 스킬
├── .mcp.json            이 프로젝트에 연결할 MCP
└── 3-folder/        🔬 매크로 — 공들이는 폴더에만 얹는 집중 규칙
```

## 쓰는 법

- `1-global/` 내용은 → 내 컴퓨터 `~/.claude/` 로 복사해서 내 것으로 고쳐 쓰기
- `2-project/` 는 → 구조를 구경한 다음, 내 프로젝트에 `.claude/` 폴더를 만들 때 참고
- 순서대로 따라 하고 싶으면 → `exercises/01-글로벌-깔기.md` 부터
