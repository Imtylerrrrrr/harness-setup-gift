# 예제 프로젝트: 단어장 퀴즈

이 프로젝트의 주인공은 코드가 아니라 **세팅 파일들**입니다.
파일이 몇 개 없으니 전부 열어보면 됩니다.

## 파일 지도

| 파일 | 층 | 역할 |
|---|---|---|
| `main.py` `words.py` | (그냥 코드) | 50줄짜리 퀴즈. 세팅을 보여주기 위한 무대 |
| `.claude/CLAUDE.md` | 🎯 프로젝트 | 이 프로젝트에서만 적용되는 규칙 |
| `.claude/skills/add-quiz-set/` | 🎯 프로젝트 | 반복작업("세트 추가")을 절차로 만든 맞춤 스킬 |
| `.mcp.json` | 🎯 프로젝트 | 이 프로젝트에 MCP(외부 도구)를 연결 |
| `3-folder/CLAUDE.md` | 🔬 폴더 | 공들이는 폴더에만 얹는 더 좁은 규칙 |

## 실행

```bash
python main.py
```

## .mcp.json 설명 (JSON은 주석을 못 다니까 여기서)

지금 들어있는 것은 **context7** — 라이브러리 최신 문서를 AI가 찾아보게 해주는 MCP입니다.
웹/파이썬 프로젝트라면 거의 항상 유용합니다.

게임(Unity) 프로젝트라면 이런 식으로 바꿔 끼웁니다:

```json
{
  "mcpServers": {
    "unity": {
      "command": "npx",
      "args": ["-y", "unity-mcp"]
    }
  }
}
```

Unity MCP는 Unity 에디터 안에서 설정하는 방식(공식)도 있으니,
자세한 것은 `exercises/03-MCP-붙여보기.md` 를 참고하세요.
