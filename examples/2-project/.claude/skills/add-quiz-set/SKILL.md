# 새 단어 세트 추가하기

이 프로젝트에서 자주 반복되는 작업 하나를 절차로 만든 "프로젝트 맞춤 스킬"입니다.
(이 프로젝트에서만 쓰니까 `프로젝트/.claude/skills/` 에 둡니다.)

## 절차

1. **주제를 확인한다.** 사용자가 주제를 안 정했으면 먼저 물어본다. (예: 중학 영단어, 역사 연도)
2. **`words.py`의 `WORD_SETS`에 새 키를 추가한다.**
   - 형식: `"세트 이름": { "문제": "정답", ... }`
   - 다른 파일을 만들지 않는다. 데이터는 무조건 `WORD_SETS` 안에.
3. **단어는 5개 이상** 넣는다. (퀴즈가 5문제를 뽑기 때문)
4. **정답은 소문자 기준**으로 쓴다. (채점이 소문자로 비교하기 때문)
5. **확인:** `python main.py`를 실행해 새 세트가 목록에 뜨고, 한 판 풀리는지 본다.

## 하지 않는 것

- main.py 수정 (세트 추가에는 필요 없다)
- 데이터 파일 분리 제안 (이 프로젝트 규모에는 과하다)
